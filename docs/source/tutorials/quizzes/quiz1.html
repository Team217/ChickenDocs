What does a robot do?

So we know what a robot is, but what does a robot do? In robotics, we are given a game for our robot to compete in. This game can vary from putting PVC pipe on an even taller PVC pipe (Reefscape 2025), to scoring rings in different places (Crescendo 2024). We create our robot to do these tasks.



How does a robot work?

For a robot to work, you first need a robot to work with. But how do you get a robot? You make it! With the help of our departments (Electrical, Mechanical, Design, Programming, Manufacturing, Business, & More), we can create our own robot. A robot works by taking code from the programmer, going through wires from electrical, and into motor put on the robot by mechanical. Basically, a robot needs many different parts to function. No one person can do it all themself!



How do I start coding?

In order to start coding, you need a few different programs first. We use WPILib as our main coding environment. Ask Adam to set it up for you. You will also need Driver Station! A full list of software you may need will be below.
